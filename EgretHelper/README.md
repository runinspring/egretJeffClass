制作egret相关内容的小工具.基于adobe AIR。请先下载运行环境。
------
### base64 图片
```
egret启动的时候可以先显示一个logo，是base64的字符串
把png图片拖拽进指示框内，就会在桌面生成一个对应的txt文件。
拷贝里面的内容到egret_loader.js替换相应内容
```
### 生成二维码图片
```
输入链接地址，点击保存按钮，会在桌面上生成 QRCode.png
支持把自定义的图片拖拽到二维码中间。中间的大小为50x50，请先处理好图片。
```
